# Dead Money Locks

Your personal NFL spread analyzer. Compares **your picks** vs **Hard Rock Bet**, finds edges, and plots patterns.

## How to Use
1. Get free API key: [the-odds-api.com](https://the-odds-api.com)
2. Go to **Settings → Secrets → Actions** → Add `ODDS_API_KEY`
3. Edit `my_spreads.csv` with your weekly picks
4. Wait for Sunday 8 PM EST → auto-run!

## Files
- `my_spreads.csv` → Your picks
- `analyze_nfl_spreads.py` → Full engine
- `full_analysis.csv` → Results
- `pattern_plot.png` → Visual edge

**You’re live!**
